# Commercial Use Guide

This is a **guide**, not the license. The license text is in `LICENSE` (PolyForm Noncommercial 1.0.0).

## Summary
- Commercial use is **not permitted** under this repository’s license.
- This guide exists to reduce “I didn’t understand what counts as commercial” ambiguity.

## Examples of commercial use (not allowed)
- Selling access to a hosted instance (SaaS).
- Including this code in a paid product or paid feature.
- Charging for consulting/installation/support **tied to** deploying or using this software.
- Monetizing via ads inside the UI (or otherwise monetizing usage).

## If you need a commercial license
- Open an issue describing your use case, or contact the maintainer via GitHub.
