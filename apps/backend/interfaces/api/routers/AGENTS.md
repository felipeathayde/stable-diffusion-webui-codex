## Purpose
- Group FastAPI routes by responsibility; each module exposes a `build_router(...)` factory.

## Modules
- `apps/backend/interfaces/api/routers/system.py` ‚Äî health/version/memory endpoints.
- `apps/backend/interfaces/api/routers/settings.py` ‚Äî settings schema + values endpoints.
- `apps/backend/interfaces/api/routers/ui.py` ‚Äî tabs/workflows/blocks/presets persistence endpoints.
- `apps/backend/interfaces/api/routers/models.py` ‚Äî model inventory + samplers/schedulers/embeddings + engine capabilities endpoints.
- `apps/backend/interfaces/api/routers/paths.py` ‚Äî `apps/paths.json` endpoints.
- `apps/backend/interfaces/api/routers/options.py` ‚Äî options store read/update/validate endpoints.
- `apps/backend/interfaces/api/routers/tasks.py` ‚Äî task status/SSE/output endpoints.
- `apps/backend/interfaces/api/routers/tools.py` ‚Äî GGUF converter + safetensors merge + file browser + PNG metadata endpoints.
- `apps/backend/interfaces/api/routers/generation.py` ‚Äî txt2img/img2img/txt2vid/img2vid/vid2vid endpoints.
- `apps/backend/interfaces/api/routers/supir.py` ‚Äî SUPIR enhance endpoints (tasks + model diagnostics).
- `apps/backend/interfaces/api/routers/upscale.py` ‚Äî upscalers inventory + remote downloads + standalone upscaling endpoints.
- `apps/backend/interfaces/api/routers/models.py` also exposes `/api/models/checkpoint-metadata` (UI metadata modal payload for a checkpoint selection).
