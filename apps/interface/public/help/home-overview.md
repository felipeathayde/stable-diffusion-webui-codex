# Codex Home — Quick Help

- **Home** is the engine-agnostic landing page. Nothing is generated here; it is a hub for creating model tabs and navigating to other workspaces.
- **Model Tabs** let you keep multiple configurations per engine:
  - SD 1.5 / SDXL / FLUX / WAN 2.2
  - You can create several WAN 2.2 tabs to target different model directories or LoRA setups.
- **Workflows** collect snapshots from tabs and expose them as reusable nodes. Use them when you want to re-run or share a configuration, not just tweak it once.
- **SDXL Workspace** is a single, opinionated image workspace. It is useful for quick runs and experiments; tabs exist when you need multiple saved variants.

When in doubt, start by creating a tab for the engine you care about, run a few generations there, and only then promote stable setups into Workflows.
